    merkle_tree v1.3.0 MerkleTree

    A hash tree or Merkle tree is a tree in which every non-leaf node is labelled with the hash of the labels or values (in case of leaves) of its child nodes. Hash trees are useful because they allow efficient and secure verification of the contents of large data structures.

    new(blocks, hash_function) :: t

    Creates a new merkle tree, given a 2^N number of string blocks and an optional hash function.

    By default, merkle_tree uses :sha256 from :crypto. Check out MerkleTree.Crypto for other available cryptographic hashes. Alternatively, you can supply your own hash function that has the spec (String.t -> String.t).
